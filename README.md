\# Employee Data Analysis using PySpark

\## Project Overview

This is a mini project for Big Data Analytics. The project analyzes an employee dataset using PySpark.

\## Dataset

The dataset used in this project is an Employee Performance dataset. It contains employee information such as:

\- ID

\- Name

\- Age

\- Gender

\- Department

\- Salary

\- Joining Date

\- Performance Score

\- Experience

\- Status

\- Location

\- Session


\## Tools Used

\- Python

\- PySpark

\- Jupyter Notebook

\- Matplotlib


\## Project Tasks

The following tasks were performed:

1\. Loaded the employee dataset.

2\. Found the average salary per department.

3\. Filtered high salary employees.

4\. Sorted employees by experience.


\## Analysis Performed

\### 1. Average Salary per Department

The dataset was grouped by department, and the average salary was calculated for each department.

\### 2. High Salary Employees

Employees with salary greater than 8000 were filtered as high salary employees.

\### 3. Employees Sorted by Experience

Employees were sorted according to their experience in descending order.


\## Visualizations

The project also includes visualizations such as:

\- Average salary per department

\- Number of employees per department

\- High salary employees per department

\- Top 10 experienced employees


\## Conclusion

This project shows how PySpark can be used to analyze employee data. It helps in understanding salary distribution across departments, identifying high salary employees, and ranking employees based on experience.
